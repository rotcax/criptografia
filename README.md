## CRIPTOGRAFIA

Proyecto institucional realizado con php y javascript donde se cifra y descifra información con los métodos de criptografia antiguos y modernos.

Se utilizan algoritmos hash y funciones php como lo son crypt y openssl.

***nota: El cifrado openssl utiliza una llave privada para dicho cifrado y se adjunta la misma en el respositorio, aunque tambien podrias generar una con openssl.***
